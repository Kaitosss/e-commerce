# E-commerce

This project is a simple E-Commerce web application that allows users to browse products, simulate an online shopping experience. The project was created for learning and practicing full-stack web development.

## Tech Stack

frontend:

- React
- Tailwind CSS
- Framer Motion

Backend:

- Express.js

Database:

- MongoDB

## Features

- Shopping cart system
- Responsive user interface
- Basic e-commerce workflow
- Admin dashboard
- Product management
